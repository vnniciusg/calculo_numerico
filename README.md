<h3>Prerequisites</h3>

- Python (3.11.6)

<h3>Cloning</h3>

```bash

```

<h3>Create a Virtual Environment</h3>

```bash
python3 -m venv venv
```

<h3>Activate the Virtual Environment</h3>


- On Windows:
```bash

```

<h3>Installing the dependencies</h3>

```bash
pip install -r requirements.txt
```
